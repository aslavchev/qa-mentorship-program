# Week 3: Functional Testing Types - Exercises

## Exercise 1: Create Smoke Test Suite (45 min)

Write 10 smoke tests for SauceDemo covering critical paths.

**Template:**
```
SMOKE-001: Application loads
SMOKE-002: Login succeeds  
SMOKE-003: Products display
SMOKE-004: Add to cart works
SMOKE-005: Cart page accessible
SMOKE-006: Checkout accessible
SMOKE-007: Logout works
SMOKE-008-010: [Add 3 more]
```

## Exercise 2: Exploratory Testing Session (60 min)

**Charter:** Explore SauceDemo shopping cart for edge cases

Document:
- Bugs found
- Questions raised
- Test ideas
- Areas of concern

## Exercise 3: Write Positive Test Cases (60 min)

Write 15 positive test cases covering:
- Login (3 tests)
- Product catalog (4 tests)
- Shopping cart (4 tests)
- Checkout (4 tests)

## Exercise 4: Write Negative Test Cases (60 min)

Write 15 negative test cases covering:
- Invalid login attempts
- Form validation errors
- Boundary conditions
- Missing required fields
- Special characters

## Exercise 5: Build Regression Checklist (30 min)

Create checklist for regression testing SauceDemo after changes.

**Deliverable:** Save all in `mentee-work/week-03/`
